# Set up

1. Clone my repositories

```
git clone https://github.com/nguyentiendat05122003/Ecommerce_f8
cd Ecommerce_f8
```

2. Install the required packages

```
npm install
```

3. Build the application:

```
npm run build
```

4. Start the development:

```
npm run dev
```
